# MConsole
A plugin for check console's output in Mobile web page.

方便你在移动端查看console的输出

# Init MConsole 初始化

```javascript
MConsole.init();
```

# Support Log 支持的 log 语句

Till now MConsole can catch console's logs as this list:
目前为止 MConsole 支持如下 console 语句：

```javascript
console.log()

console.info()

console.warn()

console.error()
```
